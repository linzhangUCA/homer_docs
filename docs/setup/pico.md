## 1. Install Thonny
[Thonny](https://thonny.org/) is a lightweight and beginner-friendly Integrated Development Environemnt (IDE) for Python.

!!! tip
    The following instructions are based on Ubuntu, but Thonny is a cross-platform software.So, you can install it on Windows, MacOS or on other OS.

Open up a terminal and execute following command:

``` sh
bash <(wget -O - https://thonny.org/installer-for-linux)
```
