# Setup Raspberry Pi Pico 2 (the Cerebellum)

## 0. Pre-Requisites
- A Raspberry Pi Pico 2 development board.
- A stack of power expansion board and Raspberry Pi (RPi) 5
- A Micro-USB cable.
- An assembled mobile base.

## 1. Install Thonny
We will use [Thonny](https://thonny.org/) to manage MicroPython on the Pico.
We recommend using official shell script to install Thonny.
On a computer (either Host or RPi), open up a terminal and execute following command:

``` sh
bash <(wget -O - https://thonny.org/installer-for-linux)
```
